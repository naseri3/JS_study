// 전체 검사            /^...$/
// 필수 포함 조건
// (?=.*\d)      // 숫자 필수
// (?=.*[a-z])   // 소문자 필수
// (?=.*[!@])    // 특수문자 필수


영문자 + 숫자
/^[a-zA-Z0-9]$/

반드시 1개 이상의 영문자 또는 숫자 사용
/^(?=.\d)[a-zA-Z0-9]+$/

문자 상관없음, 8자 이상
/^.{8,}$/

아이디 검사
/^[a-zA-Z][a-zA-Z0-9]$/

숫자로 끝난 코드
/[a-zA-Z]+\d$/

공백 없는 문자열 확인
/^\S{6,}$/

이메일 형식
/^[a-zA-Z0-9]+@[a-zA-Z0-9]+\.com$/

특수문자 포함 비밀번호
/^(?=.*[!@]).{8,}$/

숫자 개수 제한
/^\d{6,8}$/

영문자 + 숫자, 소문자 필수
/^(?=.*[a-z])[a-zA-Z0-9]{5,}$/
